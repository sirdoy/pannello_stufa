For each of the 8 files, perform the following mechanical transformation:

1. **useMemo removal:** Replace every `useMemo(() => { ... }, [deps])` or `useMemo(() => expr, [deps])` with a plain `const` assignment:
   - `const x = useMemo(() => computation, [deps])` becomes `const x = computation`
   - Multi-line bodies: unwrap the arrow function body, remove the dependency array
   - Example from useLightsData.ts: `const selectedRoom = useMemo(() => rooms.find(r => r.id === selectedRoomId) || rooms[0], [rooms, selectedRoomId])` becomes `const selectedRoom = rooms.find(r => r.id === selectedRoomId) || rooms[0]`

2. **useCallback removal:** Replace every `useCallback((args) => { body }, [deps])` with a plain `const` arrow function:
   - `const fn = useCallback((a, b) => { body }, [deps])` becomes `const fn = (a, b) => { body }`
   - `const fn = useCallback(async (a) => { body }, [deps])` becomes `const fn = async (a) => { body }` — preserve `async` keyword
   - Remove the dependency array entirely

3. **Import cleanup:** After all removals in a file, update the React import:
   - If `useMemo` was removed and no useMemo calls remain, remove `useMemo` from import
   - If `useCallback` was removed and no useCallback calls remain, remove `useCallback` from import
   - If the import becomes `import {} from 'react'` or empty, remove the entire import line
