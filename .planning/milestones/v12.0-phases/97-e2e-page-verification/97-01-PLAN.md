1. **Imports:** `import { test, expect, type ConsoleMessage, type Page } from '@playwright/test';`

2. **Helper function** `collectConsoleErrors(page: Page)` at top of file:
   - Creates `errors: string[]` array
   - Attaches `page.on('console', handler)` that pushes `msg.text()` when `msg.type() === 'error'`
   - Returns `{ errors, cleanup }` where cleanup calls `page.off('console', handler)`

3. **test.describe('Page Loads', () => { ... })** with three nested describe blocks:

   **test.describe('Dashboard', () => { ... })**:
   - `test('homepage loads with dashboard cards', ...)` [E2E-01, E2E-10]:
     - Call `collectConsoleErrors(page)` BEFORE navigation
     - `await page.goto('/')`
     - `await page.waitForLoadState('networkidle')`
     - `await expect(page.getByRole('heading', { name: 'Stufa', level: 2 })).toBeVisible({ timeout: 15000 })`
     - Call `cleanup()`, then `expect(errors).toHaveLength(0)` with message `Console errors on /: ${errors.join(', ')}`

   **test.describe('Device Pages', () => { ... })**:
   - `test('/stove loads and shows data', ...)` [E2E-02]:
     - goto `/stove`, assert `getByRole('heading', { name: 'Controllo Stufa' })` visible with timeout 15000
   - `test('/thermostat loads and shows data', ...)` [E2E-03]:
     - goto `/thermostat`
     - `await page.waitForURL(/thermostat|netatmo/, { timeout: 10000 })`
     - Assert `page.getByRole('heading').first()` visible with timeout 15000 (covers both Controllo Netatmo and Netatmo page heading)
   - `test('/lights loads and shows data', ...)` [E2E-04]:
     - goto `/lights`, assert `page.getByRole('heading').first()` visible with timeout 15000
   - `test('/network loads and shows data', ...)` [E2E-05]:
     - goto `/network`, assert `getByRole('heading', { name: 'Rete', level: 1 })` visible with timeout 15000
   - `test('/raspi loads and shows data', ...)` [E2E-06]:
     - goto `/raspi`, assert `getByRole('heading', { name: 'Raspberry Pi', level: 1 })` visible with timeout 15000

   **test.describe('Support Pages', () => { ... })**:
   - `test('/analytics loads', ...)` [E2E-07]:
     - goto `/analytics`, assert `getByRole('heading', { name: 'Analytics', level: 1 })` visible with timeout 10000
   - `test('/settings loads', ...)` [E2E-08]:
     - goto `/settings`, assert `getByRole('heading', { name: 'Impostazioni' })` visible with timeout 10000
   - `test('/debug loads', ...)` [E2E-09]:

     - goto `/debug`, assert `getByRole('heading', { name: /API Debug Console/i })` visible with timeout 10000

4. **Every test** follows the same console error pattern:
   - `const { errors, cleanup } = collectConsoleErrors(page);` BEFORE `page.goto()`
   - Content assertion in the middle
   - `cleanup(); expect(errors, \`Console errors on {path}: \${errors.join(', ')}\`).toHaveLength(0);` at end

5. **Do NOT:**
   - Import signIn/signOut (storageState handles auth)
   - Use `test.use({ storageState: ... })` (default from config is correct)
   - Use `page.waitForTimeout()` (use toBeVisible timeout instead)
   - Assert on dynamic content like stove status text
   - Add console error filters preemptively — only filter known false positives if tests fail
