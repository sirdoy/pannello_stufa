### 1. SUMMARY Frontmatter Gaps (16/25 requirements)

### 2. Nav Active-State Cosmetic Issue

Global nav items use `isActive()` with exact pathname match. When navigating to sub-pages (`/registry/devices`, `/rooms/[room_id]`), the parent nav item loses its active highlight. Device section nav items use `pathname.startsWith()` for prefix matching.

`GET /api/rooms/[room_id]/status` exists but is never called by any UI page. Phase 124 fetches all rooms at once via `GET /api/rooms/house/status`. The per-room endpoint remains available for future features.
