**useRooms hook** (inline, per D-23 through D-26):
- State: `rooms: Room[]`, `loading: boolean`, `error: string | null`
- `refetch` callback: fetches `GET /api/rooms`, on success sorts by `name.localeCompare(b.name, 'it')` (per D-06), sets rooms. On error sets error message. Always sets loading false in finally.
- `useEffect` calls `void refetch()` on mount.
- Returns `{ rooms, loading, error, refetch }`.
- NO pagination (per D-05) — no page/offset state.

**useRoomsHealth hook** (inline, per D-27 through D-29):
- State: `health: RoomsHealthResponse | null`
- `refetch` callback: fetches `GET /api/rooms/health`, on `!res.ok` silently returns, on error silently catches. Sets health on success.
- Returns `{ health, refetch }`.
