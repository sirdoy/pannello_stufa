**File 1: `lib/hooks/useWebSocketManager.ts`**

Create the singleton WebSocket connection manager per D-01, D-02, D-06, D-07, D-08.

Must include `'use client'` directive at top.

Imports:
- `useCallback, useEffect, useRef` from `'react'`
- `useWebSocket, ReadyState` from `'react-use-websocket'`
- `Topic, WebSocketMessage` from `'@/types/websocket'`

Export `TopicCallback` type: `(data: unknown) => void`

Export `WebSocketManager` interface:
```typescript
export interface WebSocketManager {
  subscribe: (topic: Topic, callback: TopicCallback) => void;
  unsubscribe: (topic: Topic, callback: TopicCallback) => void;
  readyState: ReadyState;
}
```

Re-export `ReadyState` from react-use-websocket for consumer convenience:
```typescript
export { ReadyState } from 'react-use-websocket';
```

Export function `useWebSocketManager(wsUrl: string | null): WebSocketManager`:

1. **Callback registry** — `useRef<Map<Topic, Set<TopicCallback>>>(new Map())` (per D-06).

2. **useWebSocket call** with these exact options (per D-08):
   - `onOpen`: iterate `callbacksRef.current` and for each topic with `callbacks.size > 0`, call `sendJsonMessage({ action: 'subscribe', topic })` — this implements WS-05 (re-subscribe on reconnect).
   - `shouldReconnect: () => true`
   - `reconnectAttempts: 10`
   - `reconnectInterval: (attempt: number) => Math.min(1000 * 2 ** attempt, 30000)` — implements WS-04.
   - Third argument: `wsUrl !== null` (disable connection when URL is null).

3. **Message dispatch** — `useEffect` watching `lastMessage`:
   - Parse `lastMessage.data` as `WebSocketMessage`
   - Cast `msg.topic` as `Topic`
   - Get callbacks from `callbacksRef.current.get(topic)` using optional chaining (`?.forEach`)
   - Call each callback with `msg.data`
   - Wrap in try/catch, ignore malformed messages
   - This implements WS-03.

4. **subscribe** — `useCallback`:
   - If topic not in map, create new `Set()`
   - Add callback to the set (non-null assertion safe after has-guard)
   - If `readyState === ReadyState.OPEN`, send `{ action: 'subscribe', topic }` (per D-07)
   - Dependencies: `[readyState, sendJsonMessage]`
   - This implements WS-02.

5. **unsubscribe** — `useCallback`:
   - Get callbacks set for topic via optional chaining
   - Delete the callback from the set
   - If `callbacks.size === 0` and `readyState === ReadyState.OPEN`, send `{ action: 'unsubscribe', topic }` (per D-07)
   - Dependencies: `[readyState, sendJsonMessage]`

6. Return `{ subscribe, unsubscribe, readyState }`.
