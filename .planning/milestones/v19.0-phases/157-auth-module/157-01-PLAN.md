**lib/auth/authProxy.ts** — Create function module with 4 exported functions. CRITICAL: All 4 functions use direct `fetch()` — NOT haGet/haPost/haDelete — because:
- login() requires `application/x-www-form-urlencoded` (haPost sends JSON)
- API key endpoints require `Authorization: Bearer <jwt>` (haClient hardcodes `X-API-Key`)

Internal helpers:
- `getHaBaseUrl(): string` — reads `process.env.HA_API_URL`, throws `ApiError(EXTERNAL_API_ERROR, 'HA proxy not configured: missing HA_API_URL', INTERNAL_SERVER_ERROR)` if missing
- `mapResponseError(response: Response): Promise<never>` — replicate haClient's mapResponseError logic: try to parse JSON as RFC 9457, map 401 → UNAUTHORIZED, 429 → RATE_LIMITED, 404 → NOT_FOUND, 503 → SERVICE_UNAVAILABLE, fallback → EXTERNAL_API_ERROR with BAD_GATEWAY status
- `mapCaughtError(error: unknown): never` — replicate haClient's mapCaughtError: re-throw ApiError, map AbortError → TIMEOUT, fallback → EXTERNAL_API_ERROR

Exported functions:
1. `login(username: string, password: string): Promise<Token>` — POST to `${baseUrl}/auth/login` with `Content-Type: application/x-www-form-urlencoded`, body = `new URLSearchParams({ username, password })`, no auth header (login is unauthenticated), 15s timeout via AbortController
2. `createApiKey(bearerToken: string, name: string): Promise<APIKeyResponse>` — POST to `${baseUrl}/auth/api-keys` with `Authorization: Bearer ${bearerToken}` and `Content-Type: application/json`, body = `JSON.stringify({ name })`, 15s timeout
3. `listApiKeys(bearerToken: string): Promise<APIKeyListResponse>` — GET to `${baseUrl}/auth/api-keys` with `Authorization: Bearer ${bearerToken}`, 15s timeout
4. `deleteApiKey(bearerToken: string, keyId: number): Promise<void>` — DELETE to `${baseUrl}/auth/api-keys/${keyId}` with `Authorization: Bearer ${bearerToken}`, 15s timeout. On 204: return void (no JSON parsing). On non-ok: mapResponseError.

JSDoc style: follow thermorossiProxy.ts (describe what endpoint it calls, what HA proxy does).

**app/api/auth/login/route.ts** — Create POST route handler per D-05, D-07:

```typescript
import { withAuthAndErrorHandler, success } from '@/lib/core';
import { login } from '@/lib/auth/authProxy';
import { ApiError, ERROR_CODES, HTTP_STATUS } from '@/lib/core/apiErrors';

export const dynamic = 'force-dynamic';

export const POST = withAuthAndErrorHandler(async () => {
  const username = process.env.HA_ADMIN_USER;
  const password = process.env.HA_ADMIN_PASSWORD;

  if (!username || !password) {
    throw new ApiError(
      ERROR_CODES.EXTERNAL_API_ERROR,
      'HA proxy auth not configured: missing HA_ADMIN_USER or HA_ADMIN_PASSWORD',
      HTTP_STATUS.INTERNAL_SERVER_ERROR
    );
  }

  await login(username, password);
  // Per D-03: NEVER return access_token to client
  return success({ authenticated: true });
}, 'Auth/Login');
```

**app/api/auth/api-keys/route.ts** — Create GET + POST route handlers per D-05, D-07, D-08:

```typescript
import { NextResponse } from 'next/server';
import { withAuthAndErrorHandler, success, created } from '@/lib/core';
import { login, listApiKeys, createApiKey } from '@/lib/auth/authProxy';
import { ApiError, ERROR_CODES, HTTP_STATUS } from '@/lib/core/apiErrors';
import type { APIKeyCreate } from '@/types/authProxy';

export const dynamic = 'force-dynamic';

// Helper: obtain HA proxy JWT (server-side only, per D-03)
function getAdminCredentials(): { username: string; password: string } {
  const username = process.env.HA_ADMIN_USER;
  const password = process.env.HA_ADMIN_PASSWORD;
  if (!username || !password) {
    throw new ApiError(
      ERROR_CODES.EXTERNAL_API_ERROR,
      'HA proxy auth not configured: missing HA_ADMIN_USER or HA_ADMIN_PASSWORD',
      HTTP_STATUS.INTERNAL_SERVER_ERROR
    );
  }
  return { username, password };
}

export const GET = withAuthAndErrorHandler(async () => {
  const { username, password } = getAdminCredentials();
  const { access_token } = await login(username, password);
  const data = await listApiKeys(access_token);
  return NextResponse.json(data);
}, 'Auth/ApiKeys/List');

export const POST = withAuthAndErrorHandler(async (request) => {
  const body = (await request.json()) as Partial<APIKeyCreate>;

  if (!body.name || typeof body.name !== 'string' || body.name.trim().length === 0) {
    throw ApiError.badRequest('name is required and must be a non-empty string');
  }

  const { username, password } = getAdminCredentials();
  const { access_token } = await login(username, password);
  const data = await createApiKey(access_token, body.name);
  return created(data as unknown as Record<string, unknown>);
}, 'Auth/ApiKeys/Create');
```

NOTE: GET returns `NextResponse.json(data)` instead of `success(data)` because the response is `{ keys: [...], count: N }` — an array-containing object that success() would spread incorrectly (same pattern as `app/api/rooms/route.ts` GET).

```typescript
import { withAuthAndErrorHandler, noContent } from '@/lib/core';
import { login, deleteApiKey } from '@/lib/auth/authProxy';
import { ApiError, ERROR_CODES, HTTP_STATUS } from '@/lib/core/apiErrors';

export const dynamic = 'force-dynamic';

export const DELETE = withAuthAndErrorHandler(async (_request, context) => {
  const params = await context.params;
  const keyIdRaw = params['keyId'] ?? '';
  const keyId = Number(keyIdRaw);

  // T-157-03: Validate numeric ID to prevent injection
  if (!Number.isFinite(keyId) || keyId <= 0 || !Number.isInteger(keyId)) {
    throw ApiError.badRequest('keyId must be a positive integer');
  }

  const username = process.env.HA_ADMIN_USER;
  const password = process.env.HA_ADMIN_PASSWORD;

  if (!username || !password) {
    throw new ApiError(
      ERROR_CODES.EXTERNAL_API_ERROR,
      'HA proxy auth not configured: missing HA_ADMIN_USER or HA_ADMIN_PASSWORD',
      HTTP_STATUS.INTERNAL_SERVER_ERROR
    );
  }

  const { access_token } = await login(username, password);
  await deleteApiKey(access_token, keyId);
  return noContent();
}, 'Auth/ApiKeys/Delete');
```
