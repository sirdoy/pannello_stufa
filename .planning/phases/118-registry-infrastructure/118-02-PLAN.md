**1. Create app/api/registry/types/route.ts (per D-10, D-11, D-12)**

GET is public (withErrorHandler per D-12), POST is protected (withAuthAndErrorHandler per D-11). POST returns 201 via `created()` helper.

```typescript
import { withErrorHandler, withAuthAndErrorHandler, success, created } from '@/lib/core';
import { registryProxy } from '@/lib/registry';
import type { DeviceTypeCreate } from '@/types/registry';

export const dynamic = 'force-dynamic';

/**
 * GET /api/registry/types
 * Returns all device types (built-in + custom). Public — no auth required.
 */
export const GET = withErrorHandler(async () => {
  const data = await registryProxy.getTypes();
  return success(data as unknown as Record<string, unknown>);
}, 'Registry/Types');

/**
 * POST /api/registry/types
 * Creates a custom device type. Requires authentication.
 */
export const POST = withAuthAndErrorHandler(async (request) => {
  const body = (await request.json()) as DeviceTypeCreate;
  const data = await registryProxy.createType(body);
  return created(data as unknown as Record<string, unknown>);
}, 'Registry/Types/Create');
```

**1. Create app/api/registry/devices/route.ts (per D-10, D-11, D-13)**

Both GET and POST are protected (withAuthAndErrorHandler). GET forwards pagination query params per D-13. POST returns 201 via `created()`.

```typescript
import { withAuthAndErrorHandler, success, created } from '@/lib/core';
import { registryProxy } from '@/lib/registry';
import type { DeviceCreate } from '@/types/registry';

export const dynamic = 'force-dynamic';

/**
 * GET /api/registry/devices
 * Returns paginated list of registered devices. Requires authentication.
 * Query params: limit (number), offset (number), provider_name (string)
 */
export const GET = withAuthAndErrorHandler(async (request) => {
  const sp = request.nextUrl.searchParams;
  const data = await registryProxy.getDevices({
    limit: sp.has('limit') ? Number(sp.get('limit')) : undefined,
    offset: sp.has('offset') ? Number(sp.get('offset')) : undefined,
    provider_name: sp.get('provider_name') ?? undefined,
  });
  return success(data as unknown as Record<string, unknown>);
}, 'Registry/Devices');

/**
 * POST /api/registry/devices
 * Registers a new device. Requires authentication.
 */
export const POST = withAuthAndErrorHandler(async (request) => {
  const body = (await request.json()) as DeviceCreate;
  const data = await registryProxy.registerDevice(body);
  return created(data as unknown as Record<string, unknown>);
}, 'Registry/Devices/Register');
```

CRITICAL for GET: Extract query params from `request.nextUrl.searchParams` per D-13. Use `sp.has('limit') ? Number(sp.get('limit')) : undefined` pattern — do NOT default to 0 or empty string.

**2. Create app/api/registry/devices/[device_id]/route.ts (per D-10, D-11)**

Both PUT and DELETE are protected. PUT parses JSON body as DeviceUpdate. DELETE returns 204. The `device_id` from params is a string (URL path segment) — pass as-is to proxy since the endpoint path uses it directly; the proxy's `updateDevice` and `unregisterDevice` accept `number` so convert with `Number(device_id)`.

```typescript
import { withAuthAndErrorHandler, success, noContent } from '@/lib/core';
import { registryProxy } from '@/lib/registry';
import type { DeviceUpdate } from '@/types/registry';

export const dynamic = 'force-dynamic';

/**
 * PUT /api/registry/devices/[device_id]
 * Updates a registered device's name and type. Requires authentication.
 */
export const PUT = withAuthAndErrorHandler(async (request, context) => {
  const { device_id } = await context.params;
  const body = (await request.json()) as DeviceUpdate;
  const data = await registryProxy.updateDevice(Number(device_id), body);
  return success(data as unknown as Record<string, unknown>);
}, 'Registry/Devices/Update');

/**
 * DELETE /api/registry/devices/[device_id]
 * Unregisters a device. Requires authentication.
 */
export const DELETE = withAuthAndErrorHandler(async (_request, context) => {
  const { device_id } = await context.params;
  await registryProxy.unregisterDevice(Number(device_id));
  return noContent();
}, 'Registry/Devices/Delete');
```

CRITICAL: Must `await context.params` — Next.js 15 params are Promises. Convert `device_id` to Number for proxy calls.
