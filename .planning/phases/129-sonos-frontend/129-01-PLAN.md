    **useSonosData.ts** — Follow useRaspiData.ts exactly for structure (per D-01, D-05, D-06):

    1. Create directory `app/components/devices/sonos/hooks/` and `app/components/devices/sonos/hooks/__tests__/`

    2. Define interfaces:
    ```typescript
    export interface SonosData {
      health: SonosHealthResponse;
      zones: SonosZoneResponse[];
      nowPlaying: SonosPlaybackResponse | null;
      speakerCount: number;
      zoneCount: number;
    }
    export interface UseSonosDataReturn {
      data: SonosData | null;
      loading: boolean;
      error: string | null;
      stale: boolean;
    }
    ```

    3. Export `useSonosData()` function with:
    - `useState` for data, loading (init true), error, stale
    - `useRef` for dataRef (persists across refetches)
    - `useVisibility()` for interval: visible=60000, hidden=300000
    - `fetchData` async function that:
      a. Fetches `/api/sonos/health` — parse as `SonosHealthResponse`
      b. Fetches `/api/sonos/zones` — parse as `{ zones: SonosZoneResponse[] }`, extract `.zones`
      c. Find "most interesting" zone: first with playback PLAYING, else first zone
      d. If a zone exists, fetch `/api/sonos/zones/${zone.group_id}/playback` — parse as `SonosPlaybackResponse`
      e. Build `SonosData` with `speakerCount: health.device_count`, `zoneCount: zones.length`
      f. On success: `setData`, `dataRef.current = newData`, `setStale(false)`
      g. On catch: `setStale(true)`, if `!dataRef.current` then `setError('Sonos non raggiungibile')`
      h. Finally: `setLoading(false)`
    - `useAdaptivePolling({ callback: fetchData, interval, alwaysActive: false, immediate: true, initialDelay: 600 })`
    - Return `{ data, loading, error, stale }`

    **"Most interesting zone" detail:** To determine which zone is playing, we need to fetch playback for all zones (or at least iterate). Simpler approach for the card: fetch zones, then for each zone fetch playback in parallel using `Promise.allSettled`, find first with PLAYING state, fall back to first zone's playback.

    Actually, since this is the dashboard card (not full page), optimize: fetch playback for up to 3 zones max (typical home), pick the interesting one:
    ```typescript
    const playbackResults = await Promise.allSettled(
      zones.slice(0, 5).map(z =>
        fetch(`/api/sonos/zones/${z.group_id}/playback`).then(r => {
          if (!r.ok) throw new Error('playback failed');
          return r.json() as Promise<SonosPlaybackResponse>;
        })
      )
    );
    const playbacks = playbackResults
      .filter((r): r is PromiseFulfilledResult<SonosPlaybackResponse> => r.status === 'fulfilled')
      .map(r => r.value);
    const nowPlaying = playbacks.find(p => p.transport_state === 'PLAYING') ?? playbacks[0] ?? null;
    ```
