Create `app/components/devices/sonos/components/SonosPlayModeControls.tsx`:

Props interface:
```typescript
interface SonosPlayModeControlsProps {
  playMode: SonosPlayMode | null;
  onSetPlayMode: (mode: SonosPlayMode) => void;
}
```

Implementation:
- Import `Shuffle`, `Repeat` from `lucide-react` (two functional toggles only — crossfade is NOT available in the API, omit per research pitfall 2)
- Define `decomposePlayMode(mode: SonosPlayMode | null)` helper returning `{ isShuffle: boolean; isRepeat: boolean }`:
  - `isShuffle = mode === 'SHUFFLE' || mode === 'SHUFFLE_NOREPEAT' || mode === 'SHUFFLE_REPEAT_ONE'`
  - `isRepeat = mode === 'REPEAT_ALL' || mode === 'REPEAT_ONE' || mode === 'SHUFFLE_REPEAT_ONE' || mode === 'SHUFFLE'`
    - Note: `'SHUFFLE'` means shuffle+repeat per the Sonos enum semantics
- Define `composePlayMode(currentMode: SonosPlayMode | null, toggle: 'shuffle' | 'repeat'): SonosPlayMode` helper that flips the requested toggle while preserving the other:
  - For `toggle === 'shuffle'`: if currently shuffle, remove shuffle; if not shuffle, add shuffle. Preserve repeat state.
    - `{ shuffle: true, repeat: true } → 'SHUFFLE'`
    - `{ shuffle: true, repeat: false } → 'SHUFFLE_NOREPEAT'`
    - `{ shuffle: false, repeat: true } → 'REPEAT_ALL'`
    - `{ shuffle: false, repeat: false } → 'NORMAL'`
  - For `toggle === 'repeat'`: flip repeat, preserve shuffle state. Same truth table.
- Active toggle style (per D-01): `'p-2 rounded-lg bg-ember-500/20 text-ember-400 transition-colors'`
- Inactive toggle style: `'p-2 rounded-lg bg-slate-700/50 hover:bg-slate-600/50 transition-colors text-slate-500 [html:not(.dark)_&]:bg-slate-200 [html:not(.dark)_&]:text-slate-400'`
- Render two buttons in a `<div className="flex items-center gap-2">`:
  - Shuffle button: `<Shuffle size={16} />`, aria-label="Shuffle", onClick toggles shuffle via `onSetPlayMode(composePlayMode(playMode, 'shuffle'))`
  - Repeat button: `<Repeat size={16} />`, aria-label="Ripeti", onClick toggles repeat via `onSetPlayMode(composePlayMode(playMode, 'repeat'))`
- Export as default
