### Save guard (D-14)

Save button MUST be disabled (visually muted, `cursor: not-allowed`) until:
- `draft.name.trim().length >= 1` AND
- `draft.actions.length >= 1`

Additionally, if `draft.actions` contains an `http_webhook` action with a non-empty `payload`
field that fails `JSON.parse`, an inline error row appears above the footer:
"JSON non valido" — 11px weight 600 `#ff6676` color. Save button remains blocked.

### Unsaved-changes guard (D-15)

On close attempt (Annulla, Sheet `onClose`, Escape, backdrop click), if
`JSON.stringify(original) !== JSON.stringify(draft)`:

Spawn `<ConfirmationDialog>` from `app/components/ui/ConfirmationDialog`:
- Title: "Hai modifiche non salvate. Chiudere lo stesso?"
- Primary action: "Chiudi senza salvare" — closes editor
- Secondary action: "Continua a modificare" — dismisses dialog, editor stays open

### Delete confirmation (D-16)

Edit-mode only. Clicking "Elimina" spawns `<ConfirmationDialog>`:
- Title: "Eliminare l'automazione '{name}'?"
- Primary (danger): "Elimina" — fires DELETE, closes sheet on 204
- Secondary: "Annulla" — dismisses dialog, editor stays open
