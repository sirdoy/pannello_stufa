## Tipi di Report

### Completion Report
Documenta il completamento di una feature implementata:
- Obiettivi raggiunti
- File modificati
- Test eseguiti
- Metriche di qualità

### Summary Report
Riassume un'attività di migrazione o consolidamento:
- Cambiamenti effettuati
- Breaking changes
- Note di migrazione

### Quality Analysis
Valutazione di un ResearchPack o ImplementationPlan:
- Score assegnato
- Criteri valutati
- Raccomandazioni

### Progress Report
Tracking del progresso di un workflow multi-fase:
- Fasi completate
- Stato corrente
- Blocchi/issues
